# [Unity Portfolio] 2D Idle Game
포트폴리오 제출용 유니티 팀 프로젝트 입니다.
# 프로젝트 개요
1. 프로젝트 이름 : Idle Hero
2. 프로젝트 인원 : 3명(김재훈(본인), 구학균, 김양현)
3. 프로젝트 기간 : 4주
4. 개발 언어  : C#
5. 주요 구현 분야 :
- 플레이어 스탯 시스템 기획 및 구현
- 아이템(장비) 시스템 기획 및 구현
- 모든 PopUp UI 시스템 기획 및 구현
- 퀘스트 시스템 기획 및 구현
- 저장 및 불러오기 시스템 기획 및 구현
